Hello my name is Gregory Sanchez and I am a third year student at Oregon State University.

Over the course of my time here at Oregon State I have taken a few courses of computer science. CS 162, 261, 290, 321, 344, 361, and 391.

  • I took CS 161 during my time at McMinnville Highschool where we learned everything needed to know about Java.
  • CS 162 is an introductory course for computer science where we learned all the essential components of C++.
  • CS 261 is a data structures class looking at many different search and structure ideas by using C.
  • CS 321 is an introductory course to theory of computation. This course taught compilier fundamentals with regular languages/grammer/expressions, context-free languages/grammers, and (non)recusrive languages. It showed this with DFAs, NFAs, PDAs, and turing machines.
  • CS 344 is an operating systems course learning about scripting and how it connects user programs and OS utilities together with bash, C, and python.
  • CS 361 is a software engineering course where we learned core workflow and practicies with team projects.
  • CS 391 is class looking at the social, psychological, political, and ethical issues in Computer Science.
